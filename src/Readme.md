👥 KeenKeeper — Keep Your Friendships Alive

**KeenKeeper** হলো একটি রিলেশনশিপ ম্যানেজমেন্ট ওয়েব অ্যাপ্লিকেশন যা আপনাকে আপনার বন্ধুদের সাথে নিয়মিত যোগাযোগ রাখতে সাহায্য করে। অনেক সময় ব্যস্ততার কারণে আমরা প্রিয়জনদের সাথে কথা বলতে ভুলে যাই, এই অ্যাপটি আপনাকে মনে করিয়ে দেবে কার সাথে কতদিন কথা হয়নি।

## 📝 Project Description
KeenKeeper একটি স্মার্ট ফ্রেন্ডশিপ ট্র্যাকার। এটি আপনার বন্ধুদের সাথে শেষ যোগাযোগের তারিখ ট্র্যাক করে এবং আপনার সেট করা লক্ষ্যের (Goal) ওপর ভিত্তি করে আপনাকে ফ্রেন্ডদের স্ট্যাটাস (Overdue, On-track, Almost Due) দেখায়। এটি শুধুমাত্র একটি লিস্ট নয়, বরং আপনার বন্ধুত্বের গভীরতা বজায় রাখার একটি ডিজিটাল অ্যাসিস্ট্যান্ট।

---

## 🛠️ Technologies Used
প্রজেক্টটি তৈরিতে আধুনিক সব টেকনোলজি ব্যবহার করা হয়েছে:
- **Next.js 14+** (App Router)
- **React.js** (Frontend Library)
- **Tailwind CSS** (Styling)
- **Lucide React / React Icons** (Icons)
- **Recharts** (Data Visualization)
- **React Toastify** (Notifications)
- **Local Storage API** (Data Persistence)

---

## ⭐ Key Features
১. **Smart Friendship Tracking:** বন্ধুদের কন্টাক্ট গোল অনুযায়ী অটোমেটিক স্ট্যাটাস আপডেট (Overdue/On-track) এবং রিয়েল-টাইম কাউন্টডাউন।
২. **Interactive Interaction Timeline:** প্রতিবার কল, টেক্সট বা ভিডিও চ্যাট করার পর একটি ক্লিক করলেই সেটি টাইমলাইনে সেভ হয়ে যায় এবং ফিল্টার করার সুবিধা থাকে।
৩. **Friendship Analytics:** Recharts ব্যবহার করে আপনার যোগাযোগের ধরন (Call, Text, Video) একটি সুন্দর পাই-চার্টের মাধ্যমে বিশ্লেষণ করার সুবিধা।

## 📸 Features Breakdown
- **Dynamic Navbar:** অ্যাক্টিভ পেজ ইন্ডিকেটর সহ রেসপনসিভ ন্যাভবার।
- **Quick Check-in:** ফ্রেন্ড ডিটেইলস পেজ থেকে সরাসরি কুইক অ্যাকশন লগ করার সুবিধা।
- **Sticky Layout & Loading UI:** প্রফেশনাল ইউজার এক্সপেরিয়েন্সের জন্য কাস্টম স্পিনার এবং স্টিকি ন্যাভবার।
- **Responsive UI:** মোবাইল, ট্যাবলেট এবং ডেস্কটপ — সব ডিভাইসের জন্য অপ্টিমাইজড ডিজাইন।
